# ⚛️ Disclaimer List

A DisclaimerList is an ordered list.

## Usage

```js
<script>
   import DisclaimerList, { Item } from "@dusk-network/disclaimer-list";
</script>


  <DisclaimerList>
    <Item>First</Item>
    <Item>Second</Item>
  </DisclaimerList>
```
