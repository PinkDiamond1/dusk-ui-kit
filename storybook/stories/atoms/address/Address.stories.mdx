# ⚛️ Address

Address allows users to display a wallet address.

## Variants

| Variant | Usage                                                       |
| ------- | ----------------------------------------------------------- |
| Brand   | Brand buttons are used for the principle Actions on a page. |
| CTA     | CTA buttons are used for the primary actions on a page.     |
| Info    | Info buttons are used for a passive action on a page.       |
| Success | Success buttons are used for a positive action on a page.   |
| Warning | Warning buttons are used for a risky action on a page.      |
| Danger  | Danger buttons are used for a negative action on a page.    |

## Usage

```js
<script>
    import Address from "@dusk-network/address";
    import RichText from "@dusk-network/rich-text";

    let marked = "0x66D30033B4E0BAF8970e9c8A0aD1D02Cc3e21115fhkllA9urdrTVbAyQZnwy0JLyvbCVZBHpzfBU87Gy4USFWaA6sZ";
</script>

<Address variant="{variants.ADDRESS.BRAND}">
    <RichText markdown="{marked}" />
  </Address>
```
