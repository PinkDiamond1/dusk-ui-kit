# ⚛️ Cta List

A CtaList is an ordered list.

## Usage

```js
<script>
   import CtaList, {Item} from "@dusk-network/cta-list";
</script>


  <CtaList>
    <Item>First</Item>
    <Item>Second</Item>
  </CtaList>
```
